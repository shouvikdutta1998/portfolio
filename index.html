About

I am an Offensive Security professional with 7 years of experience in Application Security, Penetration Testing, Red Teaming, and Vulnerability Assessment. Throughout my career, I've helped organizations identify and remediate security weaknesses across web applications, APIs, mobile applications, internal infrastructure, cloud environments, and Active Directory.

My professional journey began at Cyberops Infosec, followed by EY India, and I currently work as a Senior Associate at PwC AC, where I perform offensive security assessments for global clients across multiple industries.

My areas of expertise include web application security, API security, network penetration testing, mobile application security, Active Directory security assessments, cloud security reviews, and AI application security. I approach security from an attacker's perspective to uncover real-world risks that automated tools often miss — combining deep technical expertise with practical, risk-focused assessments.
